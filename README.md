# 👧 娜娜记账

一个可爱、简洁的移动端优先记账应用，帮助您轻松管理日常收支。

## ✨ 功能特点

### 📱 首页
- 本月收支概览
- 预算进度条显示
- 支出分类饼图
- 数据导出/导入功能

### ✏️ 记账页面
- 支持支出/收入切换
- 金额输入（支持小数点）
- emoji 分类选择
- 商家、地点、备注信息
- 照片上传功能
- 编辑/删除账单

### 📋 账单列表
- 按日期分组显示
- 左滑删除功能
- 导出 Excel（CSV 格式）
- 点击编辑账单

### 📊 统计页面
- 收支构成饼图
- 每日支出柱状图
- 消费 Top5 商家
- 本月/近7天切换

### 💰 预算管理
- 月度总预算设置
- 分类预算设置
- 预算进度可视化

### 🏷️ 分类管理
- 默认 12 个分类
- 自定义分类添加
- emoji 图标选择
- 颜色选择

## 🎨 设计风格

## 📁 项目结构

```
src/
├── pages/
│   ├── Home.tsx          # 首页
│   ├── AddTransaction.tsx # 记账页
│   ├── Transactions.tsx   # 账单列表
│   ├── Stats.tsx         # 统计页
│   ├── Budget.tsx        # 预算管理
│   └── Categories.tsx    # 分类管理
├── App.tsx               # 路由配置
├── store.ts              # Zustand 状态管理
├── types.ts              # TypeScript 类型定义
└── main.tsx              # 入口文件
```

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:5173 查看应用

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 🌐 部署

### Vercel（推荐）

1. 连接 GitHub 仓库
2. 自动识别 Vite 项目
3. 无需额外配置，自动部署

### 国内部署

由于 Vercel 在国内访问不稳定，推荐使用以下方案：

1. **阿里云 OSS + CDN**
2. **腾讯云 COS + CDN**
3. **Netlify**

## 💾 数据存储

应用使用浏览器 `localStorage` 存储数据，包含：
- 账单记录
- 分类配置
- 预算设置

### 数据导出/导入

- 导出：JSON 格式，包含所有数据
- 导入：支持导入之前导出的 JSON 文件

## 📱 使用技巧

1. **添加到主屏幕**
   - iOS: Safari 打开 → 分享 → 添加到主屏幕
   - Android: Chrome 打开 → 菜单 → 添加到主屏幕

2. **左滑删除**
   - 在账单列表向左滑动可删除账单

3. **导出账单**
   - 在账单列表页面点击"导出账单"按钮
   - 导出为 CSV 格式，可使用 Excel 打开
